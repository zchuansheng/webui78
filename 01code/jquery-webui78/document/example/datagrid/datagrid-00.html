数据表格插件的使用说明

名称：datagrid

定义与说明

datagrid是基于table标签实现的，对table标签定义的表格进行美化，实现需要的数据表格功能。

一、要使用datagrid，您需要完成以下工作：

1、编写table标签的html代码

2、在页面加载完成时，调用JS，完成对table的效果渲染，及参数设置

三、可选的属性

此控件可以设置的属性有

属性 类型 值 描述 设置方式

width

 数值
  • auto(缺省)
表格宽度

通过js方式设置，如：$('.grid").datagrid({width:500});

通过元素属性设置：<table width="500px">

height

 数值 500 表格宽度

通过js方式设置，如：$('.grid").datagrid({width:500});

通过元素属性设置：<table width="500px">
striped boolean
  • true(缺省)

    •  

是否使用奇偶行条纹。true:使用奇偶行条纹，flase：不使用奇偶行条纹

通过JS设置，如：$('#em_id").setValidator({min:12});

通过元素属性设置： v_min="12"
trselect boolean true(缺省)

设置表格是否可以选中某行。true:可以选中，flase：不可选中。trselect：false时defaultSel、onSelect、toolbar均无效

 通过JS设置，如：$('.grid").datagrid({trselect:false});
defaultSel boolean
  • true(缺省)
是否缺省选择第一行。true:缺省选中第一行，flase：缺省不选中任何行

通过JS设置，如：$('.grid").datagrid({defaultSel:false});
onSelect 函数对象 false(缺省)

当选中一行时的回调函数，声明onSelect(index,tr);

第1个参数是选中的行号，第2个参数是选中的tr标签对应的dom对象

function onSelectRow(index,tr){ ........}

......................

$('.grid").datagrid({onSelect:onSelectRow});
toolbar BtnToolBar对象 null 给表格绑定一个按钮工具条，当选中某行时，根据选中的数据，自动更新工具条中按钮的状态

var bar=$('.btn_toolbar').BtnToolBar();

......................

$('.grid").datagrid({toolbar:bar});
fixhead boolean false(缺省) 设置表格是否将表头固定，true固定表头，不随滚动条滚动，适用与大数据量且不分页的表格，flase：不固定表头，适用于分页或数据量较少的表格 $('.grid").datagrid({fixhead:true});
hashead boolean true(缺省) 设置是否有表头，true:有表头，flase：无表头 $('.grid").datagrid({hashead:true});
idCol 整数或字符串 -1(缺省) 序号自动增长的列编号，idCol如果是整数，则整数指定的列，将自动从1顺序标号（注意整列内容将全部清空后，设置为自增长列）。如果为字符串，将自动查找本行内匹配的class，将其text内容设置为自增长的序号。

$('.grid").datagrid({idCol:1});或

$('.grid").datagrid({idCol:'idcolClass'});
idStart 整数 1(缺省) 从第idStart行开始编号，与idCol一起使用 $('.grid").datagrid({idCol:1,idStart:3});从第三行开始自动编号
headclass 字符串 "gridheadclass"(缺省) 用来定义表头的样式名称，已经定义的样式有两个gridheadclass、gridheadclass2  
thresPage 整数 1000(缺省) 分页阀值,大于等于此数值时将自动执行前端分页  
defSizePage 整数 15(缺省) 前端分页的每页记录数  
page_message 字符串 存在未显示的数据... ... 分页导航信息的文字  
         
onAddRow(tr) 函数对象 空 当增加行时，回调函数。tr表示新增的行元素dom对象。  
colspan 数组 null 要执行相同内容合并的列号数组，从1开始 $().datagrid({colspan:[1,9]); //对第1列第9列扫描，相同单元格合并
rowspan 数组 null

要执行相同内容合并的行号数组

从1开始

 $().datagrid({rowspan:[1]}); //对第1行扫描，相同单元格合并
sortable boolean false(缺省) 表格的各列可以排序，也设置标题行的td此属性，实现对某一列的排序

1、$().datagrid({sortable:true});表格所有列都可前端排序

2、<table><tr>

<td sortable > 列标题1</td>//实现只对第一列排序

<td >列标题2</td>

</tr></table>

 

filterable boolean false(缺省) 表格的各列可以数据过滤也设置标题行的td此属性，实现对某一列的过滤

1、$().datagrid({filterable:true});表格所有列都可前端过滤

2、<table><tr>

<td filterable > 列标题1</td>//实现只对第一列过滤

<td >列标题2</td>

</tr></table>
