1、自动提交表单的示例

表单验证，一般情况下无需初始化代码。

<script type="text/javascript"> 
$(document).ready(function(){	

//一般情况下的表单验证，无需任何初始化调用
	
});		
</script>

表单元素的html代码,要如下写法，缺省情况下，可以不指定v_type,采用默认值text 

<tr>

2、自定义函数，验证、提交表单的示例

定义自己的Javascript函数

<script type="text/javascript">

//..............

function mySubmit(btn){
	//some process..........
	$.webUtil.submitOnBtn("../ok.html",$(btn));
	
}

</script>

写按钮的onclick属性，当按钮点击时，调用自定义的函数，如下：

<button align="center" class="btn" id ='btn_save' type="button"   onclick="mySubmit(this);"   >自定义函数验证提交</button>
