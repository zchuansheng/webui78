实例

图片按钮的一般用法：初始化

<script type="text/javascript"> 
$(document).ready(function(){	
	$('.btn_js').PicButton();

});		
</script>

需要显示按钮的地方，需要如下写法

<button align="center" class="btn_js"  type="button" value="1" 
 onclick="javascript:alert('some javascript code ......');return false;" >执行javascript</button>

注释： 上述代码中的class="btn" 中的class名字可以是任意自定义的类名称
